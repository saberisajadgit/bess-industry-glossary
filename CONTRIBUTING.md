# Contributing

This glossary is meant to be useful and accurate, not exhaustive. Contributions are welcome, especially:

- **Corrections** — if a definition is outdated, imprecise, or wrong, please open an issue or PR. Cite a source if you can (an ISO/RTO manual, a standards body's own documentation, etc.) — it makes the fix easy to verify and merge.
- **Missing terms** — if there's a term you think belongs here and is genuinely common in BESS engineering/commercial conversation (not just niche to one sub-specialty), suggest it via an issue.
- **Regional/market-specific terms** — this glossary currently leans US-market-heavy (PJM, CAISO, ERCOT, IRA/ITC/FEOC). Additions covering other markets (EU, Australia's NEM, etc.) are very welcome.

## What this glossary is not trying to be

- Not a textbook — entries are meant to be quick, practical, and just long enough to be useful in conversation, not exhaustive technical treatments.
- Not legal or financial advice — especially for the commercial/contract and tax-credit sections. Always verify against primary sources before relying on this for anything contractual.

## How to submit a change

1. Open a pull request editing `GLOSSARY.md` directly, or open an issue describing the suggested change if you'd rather not submit a PR.
2. Keep entries in the existing format: **Term** | one-paragraph definition, written for someone who's smart but new to the term.
3. If you're updating something that depends on a policy or market rule that changes over time (capacity-market formulas, tax-credit eligibility, etc.), note the date/source so future readers know how current it is.
